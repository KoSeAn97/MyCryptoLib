# MyCryptoLib

**Алгоритм блочного шифрования:**
* Kuznyechik
* Rijndael: AES128, AES192, AES256

**Криптографическая хэш-функция:**
* Stribog256, Stribog512
* SHA256

**Режим шифрования:**
* Cipher Feedback mode
* Output Feedback mode
* Electronic Codebook mode

**Алгоритмы Key Wrap**
* AES Key Wrap
